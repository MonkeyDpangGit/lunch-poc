登录成功

步骤1：获取access_token

获取url中access_token参数值。

步骤2：获取用户信息

调用接口，携带access_token参数，请求获取用户信息。

请求示例
curl -X GET \
'https://YOUR_TENANT_DOMAIN /auth/oauth2/userinfo' \
-H 'Authorization: Bearer {ACCESS_TOKEN} &redirect_uri= ' \
-H 'Content-Type: application/json'
